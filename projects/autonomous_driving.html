Autonomous Driving using F1/10th Car

Description:

The objective of the project is to implement Model Predictive Controller for trajectory tracking on F 1/10th car using a Jetson TX2 module. The model uncertainity will be taken into consideration while implementing the controller.

Current Progress:


  • Localization of the robot is done with particle filter algorithm using the LIDAR mounted on the robot.
  • One of the major sources of uncertainity for wheeled robots is tire slip. Quantification of the slip for various velocities is being done.
